# GPS Tracker Dashboard

A dashboard application for tracking GPS locations and displaying them on an interactive map.

## Features

- Real-time GPS location tracking
- Interactive map visualization
- Location history
- User-friendly interface

## Installation

1. Clone the repository:
```bash
git clone https://github.com/abdelazizelm/gps-tracker-dashboard.git
cd gps-tracker-dashboard
```

2. Install dependencies:
```bash
npm install
```

3. Start the application:
```bash
npm start
```
